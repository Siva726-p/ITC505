Lights Out

Project Addendum

1. Development Process

I created a 5x5 grid using HTML and styled it with CSS. JavaScript logic toggles the clicked cell and its neighbors. I ensured solvability by simulating random valid clicks on an all-white board.

2. Creative Choices

The design is minimalist and intuitive. I used black and white to clearly differentiate "on" and "off" states. The interface is mobile-friendly and simple.

3. Challenges & Solutions

Challenge: Random board setup needed to be solvable.
Solution: Instead of random toggling states directly, I simulated random valid user clicks to ensure all configurations are solvable.

Challenge: Win condition detection.
Solution: Used a loop to check if all cells are "off" (white) after each click.

4. Insights Gained

This project reinforced the importance of logic flow and modular design. I also gained more confidence in combining CSS and JS for interactive UI behavior.
